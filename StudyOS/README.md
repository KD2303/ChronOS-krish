# StudyOS

StudyOS is a study-oriented unified workspace built using the MERN stack.

## Folder Structure
- **client/**: React frontend (Vite + TailwindCSS)
- **server/**: Node.js/Express backend (MongoDB)
- **implementation_plan.md**: Detailed technical roadmap.
- **task.md**: Progress tracking checklist.
- **walkthrough.md**: Guide to running and verifying the application.

## Getting Started

1. **Install Dependencies**
   ```bash
   cd client && npm install
   cd ../server && npm install
   ```

2. **Run Backend**
   ```bash
   cd server
   npm run dev
   ```

3. **Run Frontend**
   ```bash
   cd client
   npm run dev
   ```

Refer to `walkthrough.md` for more details.
