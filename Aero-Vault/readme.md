# **Aero Vault**
It is a no-code or no-AI based IDE for student and working professional.

## **Feature**
1. Provide file explorer to manage all files
2. Live Editor
3. Complete Vs-Code theme [Black or Dark theme]
4. Provide URL manager to store important url at one place
5. Provide major snippets for coder, which include GCP, Linux, Azure, Python etc
6. User can perform CRUD operation on each feature
7. Does Provide Preview Mode

## **Tech Stack**
1. Backend ->
   Fastapi | Uvicorn | OS | Local Storage | Python 3.12 | Pydantic | RE
2. Frontend ->
   HTML,Css,JS | Vanilla JS | Google Font | Font Awesome Icon | Inter | Jet Brains Mono

## **API Endpoints**
1. GET Method -->
   /api/files
   /api/read/{filename}
   /api/search?q=query
2. POST Method -->  
   /api/save
3. DELETE Method -->
   /api/delete/{filename}
